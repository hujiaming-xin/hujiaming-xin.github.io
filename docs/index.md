# 数学老胡研究院

[文档地址](https://hujiaming-xin.github.io/)

> 没办法，数学就是这么简单。——《老胡语录·简单篇》

数学老胡研究院(Mathematics Mr.Hu Research Institute)建立于2017年，简称“胡院”，是以研究老胡及其教学内容、教学思想为研究重点的社团。由2019年6月11日正式向2020届授旗传承，后值倾覆。
